# Niche C Codebases to Know

A curated list of C-based projects that are excellent for studying architecture, minimalism, or specialized systems programming.

### 🛠 Minimalism & Philosophy
- [suckless](https://suckless.org/) - (dwm, st) Extreme minimalism; code as configuration.
- [kilo](https://github.com/antirez/kilo) - A tiny text editor in <1000 lines.
- [TinyCC (TCC)](https://bellard.org/tcc/) - A small, fast C compiler.
- [ed](https://en.wikipedia.org/wiki/Ed) - The original Unix line editor.

### ⚙️ Systems & Operating Systems
- [Linux Kernel](https://github.com/torvalds/linux) - The ultimate systems programming reference.
- [xv6](https://github.com/mit-pdos/xv6-public) - A modern re-implementation of Unix V6 for teaching.
- [FreeBSD](https://www.freebsd.org/) - Highly structured and well-documented monolithic kernel.
- [Plan 9](https://plan9.googlesource.com/plan9/) - The successor to Unix, from the same creators.
- [U-Boot](https://www.denx.de/wiki/U-Boot) - The standard bootloader for embedded systems.

### 📜 Language Runtimes & Compilers
- [Lua](https://www.lua.org/source/) - Elegant, portable, and incredibly small VM/language.
- [Python (CPython)](https://github.com/python/cpython) - The core implementation of Python.
- [GCC](https://gcc.gnu.org/) - The industry standard compiler suite.
- [MicroPython](https://github.com/micropython/micropython) - Python for microcontrollers.

### 🌐 Networking & Protocols
- [curl](https://curl.se/) - The gold standard for URL transfer.
- [nginx](https://nginx.org/) - High-performance web server and reverse proxy.
- [lwIP](https://savannah.nongnu.org/projects/lwip/) - Lightweight TCP/IP stack for embedded systems.
- [OpenSSL](https://github.com/openssl/openssl) - The foundation of web security.

### 💾 Databases & Storage
- [SQLite](https://www.sqlite.org/index.html) - The gold standard for reliability and testing.
- [Redis](https://github.com/redis/redis) - High-performance key-value store.
- [PostgreSQL](https://www.postgresql.org/) - Advanced, enterprise-grade relational database.
- [LMDB](https://github.com/LMDB/lmdb) - Lightning Memory-Mapped Database; extremely fast and small.
- [zlib](https://github.com/madler/zlib) - The industry standard for data compression.

### 🎨 Multimedia & Graphics
- [FFmpeg](https://ffmpeg.org/) - The Swiss army knife of multimedia.
- [SDL](https://www.libsdl.org/) - Simple DirectMedia Layer for cross-platform games/apps.
- [libpng](http://www.libpng.org/) - The reference implementation for PNG.
- [Pure Data](https://puredata.info/) - Visual programming language for multimedia; complex graph-based execution.

### 🛡️ Security & Cryptography
- [libsodium](https://libsodium.gitbook.io/libsodium/) - Modern, easy-to-use crypto library.
- [WireGuard](https://github.com/WireGuard/wireguard-linux) - Modern, high-performance VPN protocol.
- [qmail](https://qmail.org/) - Secure mail server design.

### 📦 Embedded & IoT
- [FreeRTOS](https://www.freertos.org/) - Widely used real-time OS.
- [Zephyr](https://www.zephyrproject.org/) - Modern RTOS for IoT.
- [BusyBox](https://busybox.net/) - The Swiss Army knife of embedded Linux.
- [musl libc](https://musl.libc.org/) - A clean, modern implementation of the C library.

### 🕹️ Game Engine & Emulation
- [Quake III Arena](https://github.com/id-Software/Quake-III-Arena) - Masterclass in 3D math and game loop architecture.
- [Raylib](https://github.com/raysan5/raylib) - Simple, hardware-accelerated graphics library.
- [QEMU](https://www.qemu.org/) - Full system and user-mode emulation.
- [MAME](https://www.mamedev.org/) - The gold standard for arcade emulation.
- [DOSBox](https://github.com/dosbox-staging/DOSBox-Staging) - Emulating x86 PC architecture.

### 🛠️ Core Utilities & Editors
- [Git](https://github.com/git/git) - Essential for studying efficient data structures (merkle trees).
- [Vim](https://github.com/vim/vim) - Complex state management and text editing.
- [Zsh](https://github.com/zsh-users/zsh) - Powerful shell implementation.
- [FUSE](https://github.com/libfuse/libfuse) - Filesystem in Userspace.

### 💎 Niche & Hidden Gems
- [stb](https://github.com/nothings/stb) - Single-header libraries for images, audio, and text.
- [libuv](https://github.com/libuv/libuv) - Asynchronous I/O engine behind Node.js.
- [jsmn](https://github.com/zserge/jsmn) - Minimalist, zero-allocation JSON parser.
- [Sokol](https://github.com/sokolsmp/sokol) - Modern, simple, cross-platform graphics/audio headers.
- [Bear](https://github.com/rizziko/Bear) - Generates compilation databases via build interception.
- [Tini](https://github.com/krallin/tini) - Tiny init process for containers.

### 🧠 AI & Low-Level Math
- [llm.c](https://github.com/karpathy/llm.c) - GPT-2 from scratch in pure C/CUDA; minimal abstraction.
- [GGML](https://github.com/ggerganov/ggml) - Tensor library for ML; foundation of llama.cpp.
- [Darknet](https://github.com/pjreddie/darknet) - Neural network framework (YOLO).

### 🌀 Weird, Experimental & Esoteric
- [IOCCC Winners](https://github.com/obfuscated/ioccc) - The peak of C obfuscation.
- [Triton](https://github.com/TritonCaldera/Triton) - Dynamic Binary Analysis framework.
- [WAMR](https://github.com/bytecodealliance/wasm-micro-runtime) - Small-footprint WebAssembly runtime.
- [Zydis](https://github.com/zydis/Zydis) - Fast x86/x64 disassembler.
